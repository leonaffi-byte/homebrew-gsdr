# homebrew-gsdr

Homebrew tap for [GSDR](https://github.com/leonaffi-byte/gsdr) — Autonomous spec-driven development for Claude Code.

## Install

```bash
brew tap leonaffi-byte/gsdr
brew install gsdr
```

## Update

```bash
brew upgrade gsdr
```

## Uninstall

```bash
brew uninstall gsdr
brew untap leonaffi-byte/gsdr
```
